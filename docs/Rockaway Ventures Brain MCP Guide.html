Rockaway Brain

Connect Claude Code or Codex to the read-only Ventures brain so the team can ask general questions in chat.

Read-only brain connection. Simple lookup skill included.

RV

What This Is

  • A read-only connection to the Rockaway Ventures brain.
  • It lets Claude Code or Codex answer questions using brain context during chat.
  • It installs a small Rockaway brain skill so teammates can ask in normal language.

What This Is Not

  • Not a meeting-prep packet generator.
  • Not an app-specific automation beyond everyday brain lookup.
  • Not a way to edit the brain.

Install On Windows

Open PowerShell and paste this command. Setup will ask for your bearer token, hidden while you type.

irm https://raw.githubusercontent.com/zabrodsk/rockaway-ventures-brain-mcp/main/setup.ps1 | iex

Install On Mac Or Linux

bash -c "$(curl -fsSL https://raw.githubusercontent.com/zabrodsk/rockaway-ventures-brain-mcp/main/bootstrap.sh)"

2How To Ask Questions

After setup, restart Claude Code or Codex if it was already open. Then ask normal questions using “Rockaway brain.” The skill will guide the agent behind the scenes.

Examples
Use the Rockaway brain to answer this: what do we know about Acme?
Use the Rockaway brain to find recent notes about this founder.
Use the Rockaway brain to find open threads around this deal.
Use the Rockaway brain to find what we know about this person.
CSV prompt
Use the Rockaway brain to enrich this CSV.

3Troubleshooting

Cannot connect

Make sure your machine is on the private network where 100.102.180.108 is reachable.

Unauthorized

Ask for a fresh Ventures bearer token and rerun the setup command.
